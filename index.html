Projets Réalisés

Infrastructure Active Directory complète pour environnement d'entreprise

Configuration Initiale du Serveur

Configuration des paramètres réseau sur Windows Server 2019, incluant l'attribution d'adresses IP statiques, la configuration DNS et la préparation pour le rôle de contrôleur de domaine.

  • Configuration IP statique (192.168.10.10)
  • Définition du masque de sous-réseau (255.255.255.0)
  • Configuration de la passerelle par défaut
  • Configuration des serveurs DNS

La configuration correcte des paramètres réseau est essentielle pour assurer le bon fonctionnement du contrôleur de domaine dans l'infrastructure.

Configuration IP via l'invite de commandes (ipconfig)

Déploiement d'Active Directory

Installation et configuration du rôle Active Directory Domain Services (AD DS) pour créer un nouvel environnement de domaine sécurisé.

  • Création du domaine SecureCorp.local
  • Configuration du contrôleur de domaine principal
  • Structure organisationnelle hiérarchique
  • Organisation par départements (IT, Finance, RH)

La conception d'une structure organisationnelle adaptée permet une gestion efficace des utilisateurs, des groupes et des stratégies de groupe.

Structure organisationnelle Active Directory avec unités d'organisation

Automatisation avec PowerShell

Développement de scripts PowerShell pour automatiser la création et la gestion des utilisateurs, des groupes et des politiques de sécurité.

  • Création de groupes de sécurité
  • Création et configuration d'utilisateurs
  • Attribution d'appartenances aux groupes
  • Définition de mots de passe sécurisés

L'automatisation avec PowerShell permet un déploiement rapide et cohérent, tout en réduisant les risques d'erreurs humaines.

Scripts PowerShell pour la création d'utilisateurs et groupes

Configuration des Stratégies de Groupe

Mise en place de stratégies de groupe (GPO) pour sécuriser l'environnement et standardiser la configuration sur l'ensemble du domaine.

  • Création de GPO globale pour l'ensemble du domaine
  • Politiques de sécurité spécifiques par département
  • Configuration des paramètres de pare-feu
  • Mise en place d'audits de sécurité

Les stratégies de groupe permettent de maintenir un niveau de sécurité élevé et cohérent sur l'ensemble des postes de l'entreprise.

Configuration des stratégies de groupe (GPO)

Intégration des Postes Clients

Configuration et intégration des postes clients Windows 10 au domaine SecureCorp.local avec vérification de l'application des stratégies.

  • Jonction des postes au domaine
  • Authentification des utilisateurs
  • Vérification de l'application des GPO
  • Tests de connexion des utilisateurs

L'intégration et les tests des postes clients permettent de valider le bon fonctionnement de l'ensemble de l'infrastructure.

Intégration d'un poste client Windows 10 au domaine
